# dotfiles

Install
=======
ln -sfv ~/.dotfiles/.bash_profile ~  
ln -sfv ~/.dotfiles/.bashrc ~  
ln -sfv ~/.dotfiles/.gitconfig ~  
ln -sfv ~/.dotfiles/.vimrc ~  
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
